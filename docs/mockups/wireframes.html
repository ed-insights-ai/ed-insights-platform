MAIN CONTENT AREA
Swaps entirely based on current page
Conference Overview / War Room / Schedule / Roster / Player Profile / etc.

Header — always visible

  • • Logo → home (conference level)
  • • Breadcrumb → current location
  • • Gender toggle → global, persistent
  • • Season selector → global
  • • User avatar → settings

Sidebar — context-aware

  • • At L1: Conference nav + team list
  • • At L2: ← Back + team identity + team nav
  • • At L3 (player): ← Back to Roster
  • • 200px fixed width
  • • Team quick-jump always reachable

Gender context bar

  • • Teal = Men's. Purple = Women's.
  • • Always visible below header
  • • Visual reminder — never ambiguous
  • • Color changes entire bar on switch
  • • Helps prevent "wrong gender" errors
